# Segmented Control

<Yard placeholderHeight={122} {...segmentedControlYardConfig} />

A segmented control is a linear set of two or more options with one segment selected at a time. A segmented control allows users to input, filter, change the presentation, or browse content in a view.

## When to use

Use a segmented control to allow users to make a single choice from up to 5 options. Segmented controls provide closely related options that allow users to input, filter, change the presentation, or browse content within a view.
